# Vedic Astrology 项目总结报告

## 项目概述

**Vedic Light** 是一个轻量版的印度星盘（吠陀占星）Web应用，专注于：
- 精准的印度星盘排盘
- AI驱动的深度解盘
- 出生时间校对功能
- 多语言支持（中文、英文、印地语）

### 核心设计理念
单一产品，单一转化路径，不做复杂功能堆砌，专注于用户体验和核心功能。

## 核心功能模块

### 1. 星盘排盘系统
**文件位置：** [`backend/app/chart_service.py`](file:///d:\Vedic%20Astrology/backend/app/chart_service.py)

**功能特性：**
- 支持完整的出生信息输入（姓名、日期、时间、经纬度、时区）
- 生成精准的吠陀星盘
- 包含 Lagna、十二宫、行星位置、Panchanga、大运、分盘摘要

### 2. AI解盘服务
**文件位置：** [`backend/app/chart_reading_service.py`](file:///d:\Vedic%20Astrology/backend/app/chart_reading_service.py)

**功能特性：**
- 集成 DeepSeek API 进行智能解读
- 支持多种解盘焦点：
  - 整体解读
  - 事业运势
  - 感情运势
- 自定义问题支持
- 多语言解盘（中文、英文、印地语）

### 3. 出生时间校对
**文件位置：** 
- [`backend/app/birth_time_rectifier_service.py`](file:///d:\Vedic%20Astrology/backend/app/birth_time_rectifier_service.py)
- [`backend/app/rectifier_dialogue_service.py`](file:///d:\Vedic%20Astrology/backend/app/rectifier_dialogue_service.py)

**功能特性：**
- AI对话式校时
- 实时查看用户星盘信息
- 根据生命事件匹配出生时间
- 多语言交互（自动适应用户语言）

### 4. 多语言支持
**文件位置：** [`frontend/types/translations.ts`](file:///d:\Vedic%20Astrology/frontend/types/translations.ts)

**支持语言：**
- 中文（简体）
- 英文
- 印地语

## 部署方案

### 本地开发
1. 后端：`cd backend && python -m uvicorn app.main:app --reload`
2. 前端：`cd frontend && npm run dev`

### Docker部署
```bash
docker-compose up -d --build
```

### 生产环境推荐
- **前端：** Vercel / Netlify
- **后端：** Railway / Render / Fly.io
